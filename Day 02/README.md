## 🎯 Different ways to display text and numbers

🔸Assigns the value 10 to variable x.
<pre> x = 10;</pre>

**<pre>disp(['The number is: ', num2str(x)])</pre>**
- ```num2str(x)``` converts number ```x``` to a string: ```'10'```
- ```['The number is: ', '10']``` concatenates the text
